#### Description
This project is a C# console application that listens to Twitch chat messages and forwards them to a specific Steam user. It's useful for streamers who want to interact with their Twitch audience directly from Steam.

#### Requirements
- .NET 6.0 or higher
- SteamKit2
- TwitchLib
- DotNetEnv
